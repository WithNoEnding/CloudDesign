# 截图清单

## 第一部分
1. 任务1：浏览器访问 `http://localhost:8080`，显示姓名“刘彬彬”和学号“2023112411”，并返回 `/api/ping` 结果。
2. 任务1：PowerShell 中 `docker compose up --build` 后端日志，能看到 `Received request: /api/ping`。
3. 任务1：SWR 控制台镜像列表，包含 `backend:v1`、`frontend:v1`。
4. 任务2：`kubectl get nodes -o wide`，至少2个 Worker 节点 Ready，含 VERSION 列。
5. 任务3：`kubectl get pods`，backend、redis、frontend Pod 均 Running。
6. 任务3：`kubectl get svc`，backend-svc 获得 EXTERNAL-IP。
7. 任务3：浏览器或 curl 访问 `http://<ELB-IP>/api/ping`，返回 `{"status":"ok"}`。
8. 任务4：`kubectl get pvc`，redis-data-pvc 状态 Bound。
9. 任务4：Redis 写入 `SET testkey hello`。
10. 任务4：删除 Redis Pod 并等待重建。
11. 任务4：重建后 `GET testkey` 返回 `hello`。
12. 任务5：`kubectl exec` 进入 frontend Pod，`cat /etc/nginx/conf.d/default.conf`，显示 ConfigMap 挂载内容。
13. 任务6：`kubectl top nodes` 有指标。
14. 任务6：`kubectl get hpa` 显示 backend-hpa。
15. 任务6：压测时 `kubectl get pods -w` 显示 backend Pod 数量增加。
16. 任务6：停止压测后 Pod 缩回。
